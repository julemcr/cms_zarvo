How to customize Scavvios easily #back to top

SASS General Options - #back to top

Scavvios is developed in SASS. by using variable it's super easy to add changes. for compiling SASS we use Gurnt OR Prepros

CSS General Options - #back to top

We have add all major styles in style.css, and it's properly commented and very easy to customize. you can pick you block at top and easily search it in style.css file

Font Options - #back to top

Scavvios HTML template has awesome features. in this template you have unlimited Font options. it's super easy and fast to change Font of complete site. we have created a variable in scss file just replace it with your Font family that's all, in result you will have you required Font family on site

Responsive Options - #back to top

Scavvios HTML template has awesome features. in this template we have created a separate css file which has all Responsive style. it's super easy and fast to change any responsive in responsive.css file.
